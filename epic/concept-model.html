Conceptual Model

User

  • userId BINARY(16) NOT NULL, Primary Key
  • userActivationToken VARCHAR (32) NOT NULL,
  • userEmail VARCHAR(64) NOT NULL,
  • userPhone VARCHAR(32) NOT NULL,
  • userHash CHAR(97) NOT NULL,
  • userFirstName VARCHAR (32) NOT NULL,
  • userLastName VARCHAR (32) NOT NULL,
  • userAge SMALLINT SIGNED,
  • userGender CHAR(8)
  • userDescription VARCHAR (140)

Animal

  • animalId BINARY(16) NOT NULL Primary Key
  • animalPhotoUrl VARCHAR (32) NOT NULL,
  • animalSpecies VARCHAR (32) NOT NULL,
  • animalName VARCHAR(32) NOT NULL,
  • animalBreed VARCHAR(32) NOT NULL,
  • animalSex VARCHAR(6) NOT NULL,
  • animalShelterId BINARY(16) NOT NULL, Foreign Key

Shelter

  • shelterId BINARY(16) Primary Key
  • shelterName VARCHAR(32) NOT NULL,
  • shelterPhone VARCHAR(10) NOT NULL,
  • shelterAddress VARCHAR(64) NOT NULL,

Like

  • LikeId BINARY(16) NOT NULL
  • UserLikeId BINARY(16) Foreign Key
  • AnimalLikeId BINARY(16) ForeignKey
